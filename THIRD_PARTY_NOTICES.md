# Avisos de terceros

Este repositorio incluye componentes que no pasan a ser propiedad del proyecto por el
hecho de distribuirse junto a él.

## Código y fuentes

- **Font Awesome 3.2.1** — fuente bajo SIL OFL 1.1; archivos CSS bajo MIT. El aviso
  original se conserva en `stylesheets/font-awesome.css`.
- **jsdom** — dependencia de desarrollo bajo licencia MIT, utilizada únicamente por
  los smoke tests. No se carga ni se distribuye con la aplicación en el navegador.

## Implementación original

El proyecto histórico contenía una referencia a `r.osey.me/code/audio` y fue importado
al historial local en 2015. La lógica JavaScript fue reemplazada por una implementación
nueva sin dependencias, pero el diseño, el CSS y las imágenes históricas todavía deben
contrastarse con la fuente original. No se encontró en este repositorio una licencia o
aviso adicional de ese origen.

## Audios y carátulas

Los archivos de demostración identifican canciones de **Sunhawk** y dos carátulas
asociadas. El repositorio no incluye permisos ni una licencia para estos recursos.
Quedan excluidos de cualquier afirmación de que el proyecto completo sea MIT hasta
que su titularidad o permiso de redistribución se documente.
